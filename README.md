# Warp
Utility to quickly setup my preferred development environment.

## Install
`curl -sL warp.ha1.io | bash -e`

## OS Support
* MacOS
* Debian / Ubuntu
* Fedora / CentOS / RHEL

## What it Installs
* Tmux
* Neovim
* Plug
* Htop

## Configuration it Loads
* Tmux conf
* vimrc

## What it Aliases
* tmux (starts with pane/window configuration)
* vim (specific neovim install)
